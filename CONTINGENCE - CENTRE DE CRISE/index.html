    À propos

    Application de Gestion de Plans de Contingence

    Cette application vous permet de créer, gérer et maintenir vos plans de contingence localement dans votre navigateur.

    Version 1.0.0

    Fonctionnalités Principales

    • Création et édition de plans de contingence structurés
    • Catégorisation par type de risque et niveau de criticité
    • Mode crise pour accès rapide aux plans critiques
    • Sauvegarde automatique via File System Access API
    • Exports multiples : JSON, CSV, PDF
    • Stockage local sécurisé (IndexedDB)
    • Interface personnalisable

    Raccourcis Clavier

    Ctrl+S Enregistrer le plan en cours
    Ctrl+F Focuser la recherche
